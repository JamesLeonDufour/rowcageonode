{% trans "About GeoNode." %}

OCHA ROWCA GeoNode is a geospatial content management system, a platform for the management and publication of geospatial data. It brings you a new way to explores The Common Operational Datasets (CODs) and The Fundamental Operational Datasets (FODs) under a consistent and easy-to-use interface allowing non-specialized users to work with those data.

Data management tools built into GeoNode allow for integrated creation of data, metadata, and map visualizations. Each dataset in the system can be shared publicly or restricted to allow access to only specific users. The Information Management Unit of OCHA ROWCA will facilitate the use, management, and quality control of the data the GeoNode instance contains.

For more information on the COD FOD please visit http://www.humanitarianresponse.info/applications/data.
